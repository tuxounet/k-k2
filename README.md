# k2 v0

K2 Build System CLI

## Basic calls

From anywhere:

```bash
npx @tuxounet-k2/cli <path to k2.inventory.yaml file> <action>
```

Allowed actions:

- create (create structure in current directory)

```bash
npx @tuxounet-k2/cli@latest create
```

- apply (apply template files)
- clean (cleanup template files)
- list
